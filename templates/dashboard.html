GET /myfxbook/events ECONOMIC CALENDAR — MYFXBOOK
DESCRIPTION

Returns scheduled economic events from the MyFxBook calendar for one or more currencies over a date range. Responses are cached for 30 minutes. Repeat the currency parameter to request multiple currencies in a single call.

EXAMPLE REQUEST URL
https://forexnews.gre8i.com/myfxbook/events?currency=CNY&currency=AUD&start_date=2026-03-10&end_date=2026-03-11
QUERY PARAMETERS
PARAMETER TYPE REQUIRED DESCRIPTION & EXAMPLE
currencyMULTI string YES
ISO 4217 currency code. Repeat the parameter to filter by multiple currencies simultaneously.
e.g. currency=USD&currency=EUR&currency=CNY
start_date string YES
Start of the date range in YYYY-MM-DD format.
e.g. start_date=2026-03-10
end_date string YES
End of the date range in YYYY-MM-DD format. Must be ≥ start_date.
e.g. end_date=2026-03-11
REQUIRED HEADERS
HEADER VALUE REQUIRED NOTES
X-API-Key sk-… YES Your API key. Alternatively pass as ?api_key=sk-… query parameter.

SHELL 
curl -X GET \
  "https://forexnews.gre8i.com/myfxbook/events?currency=CNY¤cy=AUD&start_date=2026-03-10&end_date=2026-03-11" \
  -H "X-API-Key: sk-your-api-key-here"

          GET
          /broker-spreads/live
          LIVE BROKER SPREADS — FOREXFACTORY
          
            
          
        

        

          


            

              
DESCRIPTION

              
Returns live spread and commission data for one or more brokers and trading symbols, scraped in real time from the ForexFactory broker guide. Broker names are matched case-insensitively by substring — so HFM, hfm, and HF Markets all resolve to the same broker row. Symbols accept slash or no-slash: EUR/USD and EURUSD are equivalent. Responses are cached for 5 minutes.

            


            

              
EXAMPLE REQUEST URL

              

                
                  https://forexnews.gre8i.com/broker-spreads/live?broker=HFM&symbol=EUR/USD&symbol=USD/JPY
                
                
              

            


            

              
QUERY PARAMETERS

              

                

                  
                    

                      PARAMETER
                      TYPE
                      REQUIRED
                      DESCRIPTION & EXAMPLE
                    

                  
                  
                    

                      brokerMULTI
                      string
                      YES
                      
                        
Broker name. Case-insensitive substring match against the ForexFactory broker list. Repeat to compare multiple brokers side by side.

                        
e.g. broker=HFM&broker=ICMarkets

                      		
                    

                    

                      symbolMULTI
                      string
                      YES
                      
                        
Trading symbol. Both slash and no-slash formats are accepted. Repeat for multiple symbols.

                        
e.g. symbol=EUR/USD&symbol=USDJPY

                      		
                    

                  
                

              

            


            

              
REQUIRED HEADERS

              

                

                  
                    
HEADERVALUEREQUIREDNOTES

                  
                  
                    

                      X-API-Key
                      sk-…
                      YES
                      Your API key. Alternatively pass as ?api_key=sk-… query parameter.

curl -X GET \

  -H "X-API-Key: sk-your-api-key-here"

FIELDTYPEDESCRIPTION
